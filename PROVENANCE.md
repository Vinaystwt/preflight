# Provenance

PreFlight's public repository history was sanitized before public release.

The original private working history contained operational handoffs, internal reports, rejected prototypes, local test output, deployment artifacts, and credential-adjacent notes that were not appropriate for public review. Those materials were removed from public history and preserved only in an offline local backup controlled by the operator.

The current public tree corresponds to the production source finalized on 2026-07-14. History sanitization changed commit hashes; public commit IDs are therefore not identical to the original private commit IDs.

Deployment provenance is verified against the final public `main` SHA after cutover. The npm CLI package remains `@vinaystwt/preflight-cli@0.1.0`; npm was not republished as part of the public-repository cutover.
